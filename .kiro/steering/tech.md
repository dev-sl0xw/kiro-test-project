# Technology Stack

## Core Technologies
- **HTML5**: Standard markup with semantic elements
- **CSS3**: Embedded styles with modern features (flexbox, transitions, box-shadow)
- **Korean Language Support**: UTF-8 encoding with `lang="ko"` attribute

## Build System
- No build system required - static HTML project
- Direct file serving from any web server

## Development Workflow
- Edit HTML/CSS directly in `index.html`
- Open in browser for testing
- No compilation or bundling needed

## Common Commands
```bash
# Serve locally (if Python is available)
python -m http.server 8000

# Or with Node.js
npx serve .

# Or simply open in browser
open index.html
```

## Code Style
- Embedded CSS in `<style>` tags
- Korean text content with appropriate font families
- Responsive design with viewport meta tag
- Clean, minimal styling approach
